Developer's Menu

This menu is intended solely for developers of this Forum. This menu is a guide as to how to manually login/logout of the Forum, how to store and retrieve data and how to obtain the password of users etc. Developers should refer to this menu for the sole purpose of improving the Service and if developers are found abusing the Service through this menu, they should be punished accordingly.

N.B. All actions as described below require the Console to be opened.

  1. Anonymous Login

    To perform an anonymous login (to view the Forum content, for example), simply type localStorage.setItem("pf_loggedIn", true) into the Console and refresh the page. To logout, simply click "Logout".

    Example Explained: This Forum uses the pf_loggedIn item in Local Storage to determine whether the user is logged in. By setting it to the boolean true, you are automatically logged in as an anonymous user.

  2. View Entire Database

    To view the entire account database, simply type console.log(pf_database) into the console to view the results. However, please note that this method may not be very efficient when it comes to obtaining specific account data because you then have to scan through the database to find what you were looking for.

  3. Obtain Password of Certain User (by Username)

    To obtain the password of a certain user (by Username) only, type code of the following format to the console. The code in red is the part you should actively modify depending on the situation. 

    for (i = 0; i < pf_database.length; i++) {
  if (pf_database[i].username === "<enter_username_of_user_you_want_to_fetch_the_password_of_here>") {
    console.log(pf_database[i].password);
  }
}

    Example Explained: The code above loops through the database to search for the username you have just inputted into the for loop. If it finds an account with that particular username, it logs to the console the password of that user.

  4. Login as another User (based on Username)

    To login as another user, type in the following code into the console. The parts in red denote the parts that you should actively modify to suit your situation. 

    for (i = 0; i < pf_database.length; i++) {
	if (pf_database[i].username === "<username_here>") {
		localStorage.setItem("pf_loggedIn", true);
		localStorage.setItem("pf_username", pf_database[i].username);
		localStorage.setItem("pf_password", pf_database[i].password);
		localStorage.setItem("pf_status", pf_database[i].status);
		localStorage.setItem("pf_name", pf_database[i].name);
		localStorage.setItem("pf_profilePic", pf_database[i].profilePic);
		localStorage.setItem("pf_title", pf_database[i].title);
		window.location = "index.html";
	}
}

    Example Explained: The code illustrated above loops through the database to search for the username you are looking for. If the username matches, the code logs you in with the details of that user and redirects you to index.html.
